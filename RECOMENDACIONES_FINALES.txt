RECOMENDACIONES PRIORIDAD 1 (ANTES DE PUBLICAR)
─────────────────────────────────────────────────────────────────────────────

[CRÍTICA] 1. Cambiar Contraseñas por Defecto
   Archivo: users.json
   Acción: Generar hashes SHA256 nuevos
   Impacto: Seguridad crítica

   Ejemplo Python:
   >>> import hashlib
   >>> hashlib.sha256("nuevaContraseña".encode()).hexdigest()

[CRÍTICA] 2. Crear archivo .env
   Contenido:
   - STREAMLIT_SERVER_PORT=8501
   - STREAMLIT_SERVER_HEADLESS=true
   - DATABASE_PATH=./data

   Usar: os.getenv("STREAMLIT_SERVER_PORT")

[CRÍTICA] 3. Actualizar .gitignore
   Agregar:
   - .env (credenciales)
   - users.json (base de datos de usuarios)
   - scraping_history.json
   - resultados/* (datos sensibles)
   - __pycache__/

[IMPORTANTE] 4. Implementar Validación de URLs
   En engine/scraper.py, agregar:
   - Validar formato de URL
   - Bloquear localhost, 127.0.0.1, redes privadas
   - Sanitizar selectores CSS

[IMPORTANTE] 5. Agregar Rate Limiting
   - Máximo 5 scrapes por minuto por usuario
   - Uso de decorator Python o middleware Streamlit
   - Previene abuso de recursos

[RECOMENDADO] 6. Implementar Logging de Auditoría
   - Quién ejecutó qué scraping
   - Cuándo fue
   - URL objetivo
   - Resultado (éxito/error)
